# Hexo博客搭建

# 1. Hexo

Hexo 是一个基于 Node.js 的快速、简洁的静态博客框架，适合通过 Markdown 编写内容并生成静态网页。有以下优点：

  • **支持Markdown：**文章以 Markdown 格式编写，简单易用。
  • **部署简单：**一键部署到 GitHub Pages、Netlify 等平台，与 GitHub Pages 无缝集成。
  • **主题丰富：**社区活跃，主题和插件丰富。
  • **高效快速：**利用 Node.js 高效渲染，快速生成静态页面。轻量快速，适合博客和文档。

# 2. 环境搭建

首先在官网下载并安装好node.js和git。

node.js官网：https://nodejs.org/zh-cn/download

git官网：https://git-scm.com/downloads

然后，打开cmd或者鼠标右键git bash here，搭建本地HEXO博客。

HEXO官网：https://hexo.io/zh-cn/


npm install hexo-cli -g #安装hexo命令行工具

#包管理工具
npm install -g pnpm #安装pnpm
pnpm config set registry https://registry.npmmirror.com/#阿里镜像

hexo init LZQ-CV.github.io #初始化创建
cd LZQ-CV.github.io  #进入创建的项目目录
npm install #安装依赖

hexo new test #生成test.md文件

hexo clean #清除缓存
hexo generate #生成禁态页面

#报错
npm uninstall highlight.js
npm install highlight.js


hexo server #启动本地服务器
hexo deploy #远程部署

pnpm install hexo-deployer-git --save #安装git部署插件
git init #git初始化

安装Hexo主题，官网：https://hexo.node.org.cn/themes/


git clone https://github.com/theme-next/hexo-theme-next themes/next
#主题文件放进博客根目录themes，修改theme：next

# 3.部署到GitHub


git config --global user.email ""
git config --global user.name ""
git config --global --list #查看是否配置成功
ssh-keygen -t rsa -C "email" #生成ssh公钥
ssh -T git@github.com #验证是否链接成功

hexo deploy #部署
